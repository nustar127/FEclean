# Cleaning Services E-commerce (Frontend)

## Tech Stack

- Next.js
- TypeScript
- Zustand (state management)
- Zod (form validation)
- CSS / UI styling (Tailwind CSS)

## Features

- Service catalog with list of cleaning services
- Service details page
- Add services to cart
- Cart management (update/remove items)
- Order form with validation
- Responsive layout for mobile and desktop
- Client-side state management

## Backend

This frontend communicates with a separate backend API built with Spring Boot via REST API.

## Purpose

This project was built as a practical exercise to understand real-world e-commerce flows, including cart logic, form handling, and API integration.
